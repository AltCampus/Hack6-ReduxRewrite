# Implement date-fns

Functions to implement using Date Object.

## Get Date Time

### now

It will give you the current datetime
Example:

```js
now();
// Wed Feb 13 2019 14:30:02 GMT+0530 (India Standard Time)
```

### Day of Year

Gets the day of the year.

```js
getDayOfYear(new Date());
// 44
```

### Day of Week

Gets the day of the week.

```js
getDay(new Date());
// 7
```
